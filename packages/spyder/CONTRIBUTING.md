Contributing to @funeste38/spyder

- Fork the repo and create a feature branch.
- Run `npm install` and `npm run build`.
- Add tests (vitest) where applicable and run `npm test`.
- Open a PR with a clear description. Small, focused PRs preferred.

Code style

- TypeScript with `tsc` for type checking.
- Keep dependencies minimal. Use `@funeste38/*` internal packages when relevant.
