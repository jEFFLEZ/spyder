Publishing @funeste38/spyder

1. Verify package.json (name, version, files, publishConfig.access)
2. Build: `npm run build`
3. Pack locally for verification: `npm pack` (creates `funeste38-spyder-1.0.0.tgz`)
4. Publish: `npm publish --access public`

Notes:
- Ensure account has permission to publish under `@funeste38` scope.
- Check LICENSE files before publishing.
